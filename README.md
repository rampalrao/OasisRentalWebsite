# Oasis Grandstand - Flats for Rent Website

A modern, responsive static website showcasing premium rental properties in Oasis Grandstand, Sector 22D, Yamuna Expressway.

## Features

### 🏠 Property Listings
- **Flat 1**: 2-Bedroom flat in Tower 11, Flat 204 - ₹20,000/month
- **Flat 2**: 3-Bedroom flat in Tower 10, Flat 309 - ₹18,000/month

### 📱 Responsive Design
- Fully responsive layout for desktop, tablet, and mobile devices
- Mobile-first design approach
- Touch-friendly buttons and interactions

### 🎨 Key Features
- **Direct Owner Properties**: No brokerage charges
- **Image Gallery**: Multiple high-quality images for each property with image slider
- **Virtual Tours**: YouTube video integration for property tours
- **Google Maps**: Embedded maps showing property location
- **Visit Counter**: Track page visits using localStorage
- **Contact Information**: Direct contact details with WhatsApp integration
- **Premium Amenities**: Comprehensive list of furnished amenities

### 📊 Property Details Include
- Complete amenities list
- Key features
- Property statistics (bedrooms, bathrooms, area)
- High-quality descriptions
- Contact section with WhatsApp and phone options
